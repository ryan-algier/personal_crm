# Personal CRM

A personal CRM for tracking professional networking contacts, outreach, and interactions.

Built as both a daily-use tool and a portfolio project demonstrating MySQL, Python, Excel, and Power BI skills.

## Stack

MySql - database
Python - data cleaning and anonymization
Excel - creating data
PowerBI - dashboarding 


## Setup

1. Install MySQL Community Server
2. Create a database: `CREATE DATABASE personal_crm;`
3. Run migrations in order from `sql/migrations/`
4. Install Python dependencies: `uv sync`


## Notes

No real contact data is committed to this repo.
Sample data is generated using the Faker library.
